트렌드 키워드 데이터가공정보

1. 키워드 선정 원칙

  • 해당 업종/아이템에서 시장 변화와 수요에 직접적인 영향을 주는 주제 우선
  • 제품 특성, 기술 요소, 소비 트렌드, 규제/정책 변화 등 다양한 영역을 포함
  • 데이터 수집 및 비교 가능성이 높은 키워드로 선정
  • 기본적으로 6개 내외의 핵심 키워드를 고정 유지하여 시계열 비교 가능성 확보

2. 데이터 수집 지표

기본 지표
  • 검색량 지수 (Google Trends, Naver Data Lab 등)
  • 뉴스·기사 언급량 (포털 뉴스 검색 API, 빅카인즈 등)
부가 지표(가능 시)
  • 온라인 커뮤니티 언급량 (네이버 카페/블로그, 디시인사이드, 레딧 등)
  • 주요 리포트·연구 보고서 내 키워드 출현 빈도

3. 데이터 처리 및 시각화

정규화
  • 분기별 키워드별 데이터 합계를 총합=100 기준으로 변환하여 상대 점유율 계산
절대치 분석
  • 키워드별 실제 검색량·언급량 합계를 시계열로 분석
변화량 분석
  • QoQ(전분기 대비) 및 YoY(전년 동분기 대비) 변화율 산출
시각화 구성
  • 절대치 스택 막대 + 총합 라인 → 시장 전체 관심 변화와 구성 비중 동시 파악
  • 100% 스택 막대 → 분기별 점유율 변화 확인
  • 키워드별 라인 그래프 → 개별 트렌드 추이 분석
